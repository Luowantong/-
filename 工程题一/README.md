# 工程题一的README
## 项目概述
实现一套简单的一维最优化和卡尔曼滤波库，并且用demo来检验该库是否可以正常运行
## 算法原理
**核心思想：**

1. 在预测模型所得到的值和实际观测所得到的值之间进行一定比例的选择，最终达到使得到的数据更加接近实际值
2. 因为我在预测模型只做了静止状态的模型预测，然后实际状态是匀速直线运动并且有噪声，所以最终检测结果通过调整Q R的值可以让滤波变得更加平滑，但是和实际值之间总是会有一定差距。尽管如此也可以证明这一套卡尔曼滤波库是可以使用的

**状态向量：**

在这个类里面表示状态的有协方差和状态估计值，demo测试里面只表示了当前运动路程位置

**主要公式（一维卡尔曼滤波）：**
```
x: 状态估计值
P: 误差协方差
K: 卡尔曼增益
Q: 过程噪声
R: 测量噪声
z: 测量值

预测：x=x；P=P+Q
状态更新：K=P/(P+R)；x=x+K*(z-x)；P=(1-K)*P;
```
**调试：**
在运行demo的过程中我逐渐调整Q R的取值，虽然因为预测模型是静止的导致模拟数值会有一定差距，但是Q为1.0，R为0.1的时候滤波最平滑

## demo设计思路
1. demo里面模拟的真实运动是匀速直线运动
2. 因为实际运动中有噪声，所以在demo里面利用高斯滤波来正态分布和随机数来模拟噪声
3. 像前面所说，最终预测模型和真实值有一定差距，但是这个demo只是为了证明这个库可以运行，所以没有做更加复杂的改动
