# 工程题二的README
## 项目概述
轻量cv库中的功能：
1. 图像存储
2. 图像转换：彩图转灰度图，彩图转二值化图
3. 简单图像处理算法：卷积（均值滤波），联通域
4. 图像可视化：ASCII码显示，PGM保存
### 图像存储
1. 利用std::vector来存储长、宽、通道
2. 内存布局为行优先
3. 计算寻找坐标：index=(width_*y+x)*channels_+c;

缺陷：没有处理越界访问问题
### 图像转换
1. RGB2GRAY:利用彩图通道数为3，灰度图通道数为1，并且利用加权平均法——0.299r+0.587g+0.114b
2. RGB2binary:阈值取中间值128，固定阈值做一个简单demo，但是也可以在调用的时候根据实际情况自行修改
### 图像处理

**均值滤波：**
边界像素利用0补全，防止越界

**联通域分析：**

1. 输入图像为二值图
2. 实现思路是第一遍循环所有白色区域标号，并且和上方，左边的白色区域查找是否序号相同，如果相同则该白色区域序号也一样，如果不同则比较大小，用较为小的那个
3. 第二遍循环来让每个序号都是按顺序最小的那个
4. 辅助find函数和unite函数分别用在第二次循环和第一次循环，用来查找最小标号和决定现在所循环到的标号用上面的还是左边的标号

### 图像可视化
**ASCII打印：**
运用最常用的" .:-=+*#%@"，从左到右分别是最暗到最亮

**PGM保存：**
二进制保存，内存空间占用最小
